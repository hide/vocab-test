# 英単語テスト

ブラウザで動く英単語クイズアプリ。
GitHub Pages でホスティングしています (https://hide.github.io/vocab-test/)

## 機能

### テストモード
- **読み取りテスト** — 英語を見て日本語をノートに書き、自分で正誤判定
- **書き取りテスト** — 日本語を見て英語をノートに書き、自分で正誤判定

### 出題範囲
- **全部から** — 単語リスト全体からランダムに出題
- **範囲を指定** — 「○番〜○番」の範囲からランダムに出題
- 出題数は別途設定可能（デフォルト: 10問）

### 発音機能
- 英単語の横にスピーカーボタンを表示
- クリックでブラウザの音声合成（Web Speech API）による発音を再生
- クイズ画面・答え合わせ画面・結果画面の間違えた単語一覧で利用可能

### 単語一覧表示
- 単語リスト選択メニューの横にある「一覧」ボタンで、収録単語を確認できる
- 一覧画面でも各単語の発音を確認可能

### 結果・復習
- テスト終了後にスコア（正解率・正解数・不正解数）を表示
- 間違えた単語の一覧を表示
- 間違えた単語だけで再テスト可能

## 単語リストの追加方法

1. `tests/` ディレクトリに JSON ファイルを作成:

```json
{
  "title": "リスト名",
  "words": [
    { "en": "apple", "jp": ["リンゴ"] },
    { "en": "egg", "jp": ["たまご", "卵"] }
  ]
}
```

2. `tests/index.json` にエントリを追加:

```json
[
  { "file": "new-list.json", "title": "新しいリスト" }
]
```
